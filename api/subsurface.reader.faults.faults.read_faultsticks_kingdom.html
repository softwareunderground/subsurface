read_faultsticks_kingdom

subsurface.reader.faults.faults.read_faultsticks_kingdom(fp: str, name=None)[source]

Reads in Kingdom fault stick files (kingdom) exported from Petrel (tested with Petrel 2017) and returns pandas DataFrame.

Parameters

  • fp (str) – Filepath.

  • name (str, optional) – Default: None.

Returns

(pandas.DataFrame) Fault stick information stored in dataframe with [“X”, “Y”, “Z”, “formation”, “stick id”] columns.
