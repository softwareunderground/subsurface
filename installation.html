Installation¶

$ pip install subsurface

Dependencies¶

GemPy requires Python 3.x and makes use of numerous open-source libraries:

  • pandas~=1.0.5

  • xarray~=0.16.1

  • numpy~=1.19.4

  • pytest~=6.1.2

  • setuptools~=50.3.1

  • typing~=3.7.4.3

Optional requirements:

  • segyio

  • matplotlib~=3.3.3

  • pyvista~=0.27.2

  • xlrd # For reading xlsx

  • geopandas~=0.8.1

  • rasterio~=1.1.8

  • imageio~=2.9.0

  • scipy~=1.5.4

  • shapely~=1.7.1

  • requests~=2.25.0
