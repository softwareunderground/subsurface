Parameters

  • vertex (np.ndarray) – NDArray[(Any, 3), FloatX]: XYZ point data

  • cells (np.ndarray) – NDArray[(Any, …), IntX]: Combination of vertex that create different geometric elements

  • attributes (pd.DataFrame) – NDArray[(Any, …), FloatX]: Number associated to an element

  • points_attributes (pd.DataFrame) – NDArray[(Any, …), FloatX]: Number associated to points

  • ds (xarray.Dataset) – Directly a dataset with the expected structured. This arg is specially thought for loading data from disk

Notes

Depending on the shape of edge the following unstructured elements can be create: - cells NDArray[(Any, 0), IntX] or NDArray[(Any, 1), IntX] -> Point cloud.

E.g. Outcrop scan with lidar

  • cells NDArray[(Any, 2), IntX] -> Lines. E.g. Borehole

  • cells NDArray[(Any, 3), IntX] -> Mesh. E.g surface-DEM Topography

  • cells NDArray[(Any, 4), IntX]

    • -> tetrahedron

    • -> quadrilateral (or tetragon) UNSUPPORTED?

  • cells NDArray[(Any, 8), IntX] -> Hexahedron: Unstructured grid/Prisms
